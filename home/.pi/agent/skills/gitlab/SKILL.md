# GitLab via glab CLI

Use `glab` for all GitLab interactions. It's installed and authenticated.

```bash
glab --help          # full command list
glab <cmd> --help    # subcommand details
```

Common commands: `glab mr`, `glab issue`, `glab pipeline`, `glab repo`.
